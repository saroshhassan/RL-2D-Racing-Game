A **2D racing game** built in Python using **Pygame**. This project demonstrates integration of classic game mechanics with reinforcement learning concepts. The player competes against a CPU-controlled car powered by an RL agent, racing on predefined tracks while avoiding obstacles and tracking performance.  

---

## Features  

- **Player Car** ‚Äì Control with keyboard (arrow keys or WASD).  
- **CPU Car** ‚Äì Driven by a reinforcement learning (RL) agent.  
- **Track** ‚Äì Includes start/finish lines with a grey boundary to avoid.  
- **Timer System** ‚Äì Displays race time for both cars.  
- **Collision Detection** ‚Äì Cars passing into grey areas take damage.  

---

## Controls  

- **Arrow Keys / WASD** ‚Üí Steer & accelerate the player car  
- **Q** ‚Üí Quit the game
